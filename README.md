# AI-Practice
## 인공지능 모델 개발 절차
1. **문제 정의** : 해결하고자 하는 문제 정의. 목표, 성능 지표 설정
2. **데이터 수집 및 준비** : 해결에 필요한 데이터 수집, 정제, 전처리, 특성 공학 수행. 학습/검증/테스트용 데이터 분리
3. **모델 선택 및 설계** : 문제 유형에 맞는 모델(분류-Decision Tree, 이미지-CNN, 시퀀스-RNN/Transformer) 선택, 모델 구조(층, 노드 수) 설계
4. **모델 학습** : 학습 데이터로 모델 훈련. 모델은 가중치와 편향 조정
5. **모델 평가 및 튜닝** : 검증, 테스트 데이터로 성능 평가. 불만족 시 하이퍼파라미터(학습률, 배치 크기 등) 조정 후 재학습
6. **배포** : 실제 서비스 환경에 적용
