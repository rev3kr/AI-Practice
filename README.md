# AI-Practice
## 인공지능 모델 개발 절차
1. **문제 정의** : 해결하고자 하는 문제 정의. 목표, 성능 지표 설정
2. **데이터 수집 및 준비** : 해결에 필요한 데이터 수집, 정제, 전처리, 특성 공학 수행. 학습/검증/테스트용 데이터 분리
3. **모델 선택 및 설계** : 문제 유형에 맞는 모델(분류-Decision Tree, 이미지-CNN, 시퀀스-RNN/Transformer) 선택, 모델 구조(층, 노드 수) 설계
4. **모델 학습** : 학습 데이터로 모델 훈련. 모델은 가중치와 편향 조정
5. **모델 평가 및 튜닝** : 검증, 테스트 데이터로 성능 평가. 불만족 시 하이퍼파라미터(학습률, 배치 크기 등) 조정 후 재학습
6. **배포** : 실제 서비스 환경에 적용

## 분류 (Classification)
머신러닝 핵심 분야 중 하나로,<br>
**주어진 데이터를 정해진 카테고리 중 하나로 예측하고 구분하는 작업**<br>
분류는 정답 카테고리가 몇 개인지 따라 크게 두 가지로 나뉨.
1. 이진 분류 : 정답 카테고리가 2개인 경우
2. 다중 분류 : 정답 카테고리가 3개 이상인 경우

## 회귀 (Regression)
**데이터에 가장 잘 맞는 선(추세선) 찾기**<br>
데이터들을 가장 잘 설명할 수 있는 최적의 선을 찾고, 이를 이용해 연속적인 값을 예측

## LLM (Large Language Model)
**수많은 양의 텍스트 데이터를 학습한 인공지능 모델**<br>
방대한 학습 - 문맥 파악 - 지식 내재화
### LLM 핵심 능력
1. 택스트 생성
2. 텍스트 이해
3. 대화
